# Project Management System

A full stack project management application built using React.js, Node.js, Express, and MongoDB.

## Features

* User Authentication (Login/Register)
* Project Management
* Task Management
* Status Tracking
* Dashboard
* JWT Authentication

## Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB
* JWT
* CSS and Material UI
